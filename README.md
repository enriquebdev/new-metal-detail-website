## 📈 Evolução do projeto

Este projeto foi desenvolvido de forma gradual, acompanhando minha evolução em HTML, CSS e JavaScript. Durante o desenvolvimento, novas funcionalidades foram adicionadas, problemas foram identificados e o código passou por melhorias para se tornar mais organizado, responsivo e funcional.

### Primeiros passos

- Criação da estrutura inicial da página utilizando HTML.
- Organização das principais seções do site.
- Definição da estrutura dos serviços, formulário de contato e apresentação da empresa.

### Construção da interface

- Desenvolvimento do layout utilizando CSS.
- Criação da identidade visual do site com cores, tipografia e organização dos elementos.
- Implementação de um design responsivo para diferentes tamanhos de tela.
- Ajustes de espaçamento, posicionamento e adaptação dos componentes para dispositivos móveis.

### Adicionando interatividade

- Implementação das funcionalidades utilizando JavaScript.
- Manipulação do DOM para controlar elementos da página.
- Desenvolvimento de um carrossel de serviços responsivo utilizando JavaScript puro.
- Implementação de movimentação dinâmica baseada no tamanho real dos elementos através do `offsetWidth`.
- Criação de um sistema de carrossel infinito utilizando clonagem de elementos.
- Controle de animações e bloqueio de múltiplos cliques para evitar conflitos durante a transição.
- Ajustes no comportamento do carrossel para diferentes resoluções utilizando eventos de redimensionamento.
- Integração do formulário de contato utilizando Sheet Monkey, permitindo o envio automático das informações para uma planilha do Google Sheets.

### Melhorias e organização do código

- Refatoração do JavaScript para melhorar a organização e facilitar futuras alterações.
- Separação das responsabilidades dentro do código, deixando a lógica mais clara.
- Correção de problemas relacionados à responsividade, posicionamento de elementos e transições.
- Aprendizado sobre controle de estado, eventos e atualização dinâmica de componentes.

### Situação atual

- Site totalmente responsivo.
- Carrossel de serviços funcional e adaptável para diferentes dispositivos.
- Código organizado para facilitar manutenção e expansão.
- Projeto desenvolvido para um negócio real, aplicando programação para resolver uma necessidade prática.
- Estrutura preparada para receber novas funcionalidades conforme minha evolução nos estudos.
