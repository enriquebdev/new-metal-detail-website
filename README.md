## 📈 Evolução do projeto

Este projeto foi desenvolvido de forma gradual, acompanhando minha evolução em HTML, CSS e JavaScript. A cada etapa, novos recursos foram adicionados e o código foi sendo reorganizado para deixá-lo mais limpo e funcional.

### Primeiros passos
- Estrutura inicial da página em HTML.
- Organização das seções principais do site.

### Construção da interface
- Desenvolvimento do layout utilizando CSS.
- Definição das cores, tipografia e identidade visual.
- Ajustes para diferentes tamanhos de tela.

### Adicionando interatividade
- Implementação das funcionalidades com JavaScript.
- Desenvolvimento do carrossel de serviços.
- Manipulação do DOM.
- Integração do formulário de contato utilizando Sheet Monkey, permitindo o envio automático das informações para uma planilha do **Google Sheets**.

### Situação atual
- Site responsivo.
- Código organizado para facilitar futuras melhorias.
- Projeto desenvolvido para um negócio real, unindo prática de programação com uma necessidade da empresa.
- Estrutura preparada para receber novas funcionalidades conforme minha evolução nos estudos.
