📈 Evolução do projeto

Este projeto foi desenvolvido de forma gradual, acompanhando minha evolução em HTML, CSS e JavaScript. Durante o desenvolvimento, novas funcionalidades foram adicionadas, problemas foram identificados e o código passou por melhorias para se tornar mais organizado, responsivo e funcional. O repositório também conta com capturas de tela e uma demonstração em GIF para apresentar o funcionamento da aplicação.

Primeiros passos
Criação da estrutura inicial da página utilizando HTML.
Organização das principais seções do site.
Definição da estrutura dos serviços, formulário de contato e apresentação da empresa.
Construção da interface
Desenvolvimento do layout utilizando CSS.
Criação da identidade visual do site com definição de cores, tipografia e organização visual.
Implementação de um design responsivo para diferentes tamanhos de tela.
Ajustes de espaçamento, posicionamento e adaptação dos componentes para dispositivos móveis.
Adicionando interatividade
Implementação das funcionalidades utilizando JavaScript.
Manipulação do DOM para controlar os elementos da página.
Desenvolvimento de um carrossel de serviços responsivo utilizando JavaScript puro.
Implementação de movimentação dinâmica baseada no tamanho real dos elementos através do offsetWidth.
Criação de um sistema de carrossel infinito utilizando clonagem de elementos.
Controle das animações e bloqueio de múltiplos cliques para evitar conflitos durante as transições.
Ajustes automáticos do carrossel para diferentes resoluções utilizando eventos de redimensionamento.
Integração do formulário de contato utilizando Sheet Monkey, permitindo o envio automático das informações para uma planilha do Google Sheets.
Melhorias e organização do código
Refatoração do JavaScript para tornar a estrutura mais organizada e de fácil manutenção.
Separação das responsabilidades do código para melhorar a legibilidade.
Correção de problemas relacionados à responsividade, posicionamento de elementos e transições.
Aprendizado sobre controle de estado, eventos, manipulação do DOM e atualização dinâmica de componentes.
Organização do projeto e da documentação para facilitar futuras evoluções.
Documentação do projeto
Inclusão de capturas de tela das principais seções do site.
Adição de um GIF demonstrando o funcionamento da aplicação.
README atualizado com descrição das funcionalidades, tecnologias utilizadas e evolução do desenvolvimento.
Situação atual
Site totalmente responsivo.
Carrossel de serviços funcional e adaptável para diferentes dispositivos.
Formulário de contato integrado ao Google Sheets.
Código organizado para facilitar manutenção e expansão.
Projeto desenvolvido para um negócio real, aplicando programação para resolver uma necessidade prática.
Documentação enriquecida com imagens e demonstração visual do funcionamento.
Estrutura preparada para receber novas funcionalidades conforme minha evolução nos estudos.
